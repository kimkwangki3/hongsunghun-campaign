# 선거사무관계자 실비 스킬
> 출처: 공직선거법 제135조, 공직선거관리규칙 제59조, 공무원여비규정 별표2, 헌법재판소 2015.4.30. 선고 2013헌바55 결정  
> 기준: 2025년 10월 1일 시행 공직선거법 (법률 제21066호)  
> 대상: 2026년 6월 3일 제9회 전국동시지방선거 전라남도의회의원 순천시 제7선거구 출마 캠프

---

## 1. 실비의 법적 정의

공직선거법 제135조 및 헌법재판소 결정(2013헌바55) 기준:

> **실비 = "선거사무관계자가 선거운동과 관련하여 실제로 소요된 비용"**

헌법재판소는 실비의 종류를:
- **교통비**
- **식사비**
- **기타 비용**

으로 명시하고 있다.

---

| 법령 | 조항 | 내용 |
|---|---|---|
| 공직선거법 | 제135조 제1항 | 선거사무관계자에게 수당과 실비 지급 가능 |
| 공직선거법 | 제135조 제5항 | 실비의 종류와 금액은 중앙선거관리위원회규칙으로 정함 |
| 공직선거관리규칙 | 제59조 | 실비 기준: 공무원여비규정 별표2 제2호 준용 |
| 공무원여비규정 | 별표2 제2호 | 일비·식비 기준 (숙박비 제외) |

---

## 3. 지역구 시·도의원선거 선거사무관계자 실비 기준

공직선거관리규칙 제59조 제4호 기준:

> **지역구 시·도의원선거의 선거사무장**: 5만원 이내 수당 + 공무원여비규정 별표2 **제2호** 실비  
> **선거사무원**: 3만원 이내 수당 + 공무원여비규정 별표2 **제2호** 실비

### 공무원여비규정 별표2 제2호 기준 (숙박비 제외)

| 항목 | 기준 | 비고 |
|---|---|---|
| **일비** | 1일 **20,000원** | 출장지 소요 각종 비용 충당용 |
| **식비** | 1일 **25,000원** | 실제 식사 여부와 무관하게 정액 |
| **숙박비** | **지급 불가** | 선거사무관계자 숙박비 명시 금지 |

> ※ 위 금액은 공무원여비규정 별표2 제2호(5급 이하 공무원 등) 기준이며  
> 규칙 개정 시 변경될 수 있으므로 **선관위 최신 공고 확인 필수**

---

## 4. 실비 항목 상세

### 4-1. 일비 (1일 20,000원)
- 출장지에서 소요되는 각종 잡비 포함
- 교통비, 통신비, 기타 소액 비용 포괄
- 별도 영수증 제출 없이 정액 지급 가능

### 4-2. 식비 (1일 25,000원)
- **식사 제공 시 반드시 공제**
- 공제 기준 (공직선거관리규칙 제59조 제2항):

| 상황 | 공제액 | 지급액 |
|---|---|---|
| 식사 미제공 | 0 | 25,000원 |
| 식사 1끼 제공 (8,330원 상당) | 8,330원 | 16,670원 |
| 식사 2끼 제공 (16,660원 상당) | 16,660원 | 8,340원 |
| 식사 3끼 제공 (25,000원 상당) | 25,000원 | 0원 |

> ※ 식사비 공제 없이 식사 제공 + 실비 전액 지급 = **이중지급 → 위법**

### 4-3. 교통편의 제공 시
- 교통편의를 제공한 경우 **실비에서 해당 금액 공제**
- 교통편의 금액 공제 후 잔액만 지급

## 6. 1일 최대 지급 가능 금액

### 선거사무장 (식사 미제공 기준)
```
수당 100,000원 + 일비 20,000원 + 식비 25,000원 = 145,000원/일
```

### 선거사무원 (식사 미제공 기준)
```
수당 60,000원 + 일비 20,000원 + 식비 25,000원 = 105,000원/일
```

### 선거기간 13일 기준 총액

| 직책 | 1일 최대 | 13일 합계 | 선거비용 보전 |
|---|---:|---:|---|
| 선거사무장 | 145,000원 | 1,885,000원 | ✅ |
| 회계책임자 | 145,000원 | 1,885,000원 | ✅ |
| 선거사무원 | 105,000원 | 1,365,000원 | ✅ |

## 7. 절대 금지 사항

| 금지 항목 | 근거 |
|---|---|
| 숙박비 지급 | 공직선거관리규칙 제59조 (숙박료 제외 명시) |
| 식사 제공 후 식비 공제 없이 전액 지급 | 이중지급 → 위반 |
| 교통편의 제공 후 일비 공제 없이 전액 지급 | 이중지급 → 위반 |
| 신고되지 않은 자에게 수당·실비 지급 | 기부행위 또는 매수 |
| 법정 기준 초과 지급 | 공직선거법 제135조 위반 |
| 자원봉사자에게 실비 명목 지급 | 명목 불문 금지 |

---

## 8. 지급 방법 및 증빙

### 지급 방법
- 반드시 **계좌이체**
- 신고된 지출 계좌에서 선거사무관계자 본인 계좌로 이체
- 본인 명의 계좌 없을 시 → 가족 명의 계좌 + 소명서 첨부

### 필수 증빙
- 수당·실비 **지급명세서** (날인 또는 서명 포함)
- 계좌이체 내역
- 선거사무관계자 **선임 신고 자료**
- 식사 제공 여부 기록 (공제 근거)
- 교통편의 제공 여부 기록 (공제 근거)

| 후보자+동행자 식사 | 공직선거법 선거비용 규정 | 별도 선거비용으로 처리 |

## 12. 핵심 요약

```
실비 = 교통비(일비) + 식비 (숙박비 제외)
일비 = 1일 20,000원 (정액)
식비 = 1일 25,000원 (식사 제공 시 공제)
숙박비 = 지급 불가
식사 제공 + 식비 전액 지급 = 이중지급 위법
지급 방법 = 계좌이체 필수
```
